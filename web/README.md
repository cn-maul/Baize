# 白泽(Baize)前端界面使用指南

## 概述

本目录包含白泽(Baize) AI网关的前端界面，用于测试和使用AI模型聚合网关的API。前端界面使用HTML5、Tailwind CSS v3和JavaScript构建，提供了一个直观、美观的用户界面。

## 文件结构

```
web/
├── index.html          # 前端主页面
└── README.md           # 本使用指南
```

## 使用方法

### 1. 准备工作

1. **启动后端服务**：
   - 在项目根目录运行：`go run cmd/baize/main.go`
   - 确保服务运行在 `http://localhost:8080` 或您指定的端口

2. **配置API地址**（如果需要）：
   - 编辑 `index.html` 文件，修改 `API_BASE_URL` 变量
   - 默认值：`http://localhost:8081/api/v1`

### 2. 打开前端界面

在浏览器中打开以下地址：

```
file:///c:/Users/dogge/Desktop/Baize/web/index.html
```

### 3. 使用界面

#### 3.1 配置区域

- **选择平台**：从下拉菜单中选择一个AI平台（如OpenAI、Anthropic）
- **选择模型**：选择平台后，模型下拉菜单会自动加载该平台的可用模型

#### 3.2 对话区域

- **消息输入**：在底部输入框中输入消息
- **发送消息**：点击"发送"按钮或按回车键发送消息
- **查看回复**：AI的回复会显示在聊天历史区域
- **聊天历史**：显示所有的对话记录

#### 3.3 信息区域

- **API信息**：显示API接口地址和服务状态
- **平台信息**：显示所有可用平台及其模型列表

## 功能特点

1. **响应式设计**：适配不同屏幕尺寸，在手机和桌面端都能良好显示
2. **实时状态**：顶部状态栏显示服务连接状态
3. **加载动画**：平台列表加载时显示动画效果
4. **错误提示**：当操作错误时显示友好的错误提示
5. **美观界面**：采用现代化的设计风格，使用Tailwind CSS构建
6. **完整功能**：支持选择平台、选择模型、发送消息、查看回复等所有核心功能

## 常见问题

### Q: 前端无法加载平台列表

**A:** 请检查：
1. 后端服务是否正在运行
2. 前端的API地址是否正确
3. 浏览器控制台是否有错误信息
4. 网络连接是否正常

### Q: 发送消息后没有回复

**A:** 请检查：
1. 平台和模型选择是否正确
2. API密钥是否有效
3. 网络连接是否正常
4. 后端服务日志是否有错误信息

### Q: 界面显示异常

**A:** 请尝试：
1. 刷新浏览器页面
2. 清除浏览器缓存
3. 使用最新版本的Chrome、Firefox或Edge浏览器

## 技术实现

- **HTML5**：页面结构
- **Tailwind CSS v3**：样式和响应式设计
- **JavaScript**：交互逻辑
- **Font Awesome**：图标
- **Fetch API**：与后端服务通信

## 自定义

### 修改API地址

编辑 `index.html` 文件，找到以下代码并修改：

```javascript
// API基础URL
const API_BASE_URL = 'http://localhost:8081/api/v1';
```

### 修改样式

编辑 `index.html` 文件中的Tailwind配置：

```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                primary: '#3b82f6',
                secondary: '#64748b',
                accent: '#10b981',
                dark: '#1e293b',
                light: '#f8fafc'
            },
            fontFamily: {
                sans: ['Inter', 'system-ui', 'sans-serif']
            }
        }
    }
}
```
